Title: How to Install Jenkins on Ubuntu [Step-by-Step Tutorial]

Introduction:
Welcome to this tutorial on installing Jenkins on Ubuntu. Jenkins is a popular automation server used for continuous integration and continuous delivery (CI/CD) pipelines. In this tutorial, we'll walk through each step required to install Jenkins on Ubuntu, ensuring you understand what each command does along the way.

Step 1: Update Package Lists
First, let's make sure our package lists are up to date by running the following command:

sudo apt update

This command updates the local package index to ensure you have the latest information about available packages and their versions.

Step 2: Install OpenJDK 11
Jenkins requires Java to run. We'll install OpenJDK 11 using the following command:

sudo apt install openjdk-11-jdk -y

This command installs the OpenJDK 11 development kit, which includes the Java Runtime Environment (JRE) and the Java Development Kit (JDK).

Step 3: Verify Java Installation
After installing Java, let's verify the installation by checking the Java version:

java -version

This command will display the version of Java installed on your system.
